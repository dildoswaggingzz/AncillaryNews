Historical backfill

The ingestor's scheduled poller only ever fetches the most recent handful of records per dataset, so market_data_history only has real depth back to whenever it was first brought up. Use this to backfill weeks of real history for the datasets the BESS backtest simulator (/dashboard/bess/new) reads, via Energinet's public start/end date-range API. Free -- pure Energinet HTTP polling + Postgres writes, no Anthropic API calls. Safe to re-run (see shared/backfill.py).
